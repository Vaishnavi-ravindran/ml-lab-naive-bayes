# Naive Bayes Classifier from Scratch

This project implements the Naive Bayes classification algorithm from scratch using Python.

## Features
- Prior probability calculation
- Likelihood estimation with Laplace smoothing
- Posterior probability computation
- Classification of new samples
- Accuracy evaluation on dataset

## Dataset
Custom CSV dataset (data.csv)

## Output
- Predicted class for input samples
- Posterior probabilities
- Training accuracy

## Tools Used
- Python
- Pandas

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Open the notebook:
   naive_bayes_classifier.ipynb

## Concepts Covered
- Naive Bayes Theorem
- Probability Theory
- Laplace Smoothing
- Classification
